Game 

Это игра при которой выбераються две команды и персонажи к ним 
Макс колличество 4 персонажа(можно выбирать одного и того же хоть 4 раза)
Каждому даеться определенное количество жизни/здоровья 
Затем можно выбрать оружие/эффект по желанию 
Все персонажи примерно равны а характеристики не известны для пользателей 
Игра идет на консоли и описывають каждое действие персонажа
До победы одной из команд 

Игра включает в себя:
Уроны по здороаью персонажа 
Процентность урона
Каждое действие и последствия описываються
Игра идет автомотический и пользовотель видит процесс и уже результат игры 

Персонажи: 
Персонажа типа "Танк"
Танк имеет 5 начальных очков во всех характеристиках (например, сила, ловкость, защита, скорость)
У Танк будет выбран один из трех навыков: "Ответная Атака", "Железная защита" или "Таунт"
В этом примере Танк выбирает навык "Железная защита", который усиливает его защитные способности

Персонажа типа "Ассасин"
Ассасин также начинает с 5 очками во всех характеристиках
Ассасин может выбрать один из следующих навыков: "Ядовитый клинок", "Теневой шаг" или "Мастер убийства"
В этом примере Ассасин выбирает "Ядовитый клинок", который добавляет ядовитый урон врагу

Персонажа типа "Маг"
Маг будет иметь начальные характеристики 5 для всех параметров
Он может выбрать один из навыков: "Магический барьер", "Электрошок" или "Файрбол"
В примере Маг выбирает "Файрвол", который защищает его от урона

Персонажа типа "Снайпер"
Снайпер, как и другие персонажи, начинает с 5 очками в каждой характеристике
Он может выбрать один из трех навыков: "Оглушающая пуля", "Смертельный выстрел" или "Двойной Удар"
В данном примере Снайпер выбирает "Оглушающую пулю", чтобы оглушить врагов
